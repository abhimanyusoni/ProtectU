Abstract

ProtectU is an OnDemand Security Services Booking Platform. Organization is a specialized brand of professional private security services encompassing all industries. With their Digital platform they are providing new avenues of security service delivery. All their professional services are delivered in a professional, expedited, and discreet manner.

CMARIX has designed and developed ProtectU platform which includes Native iOS App, Native Android App, Web Platform as well as custom Admin Panel. Client’s Requirement

Client is a domain expert in security services with more than 25+ years of experience. They had a clear vision to build an OnDemand application to provide different security services for anyone and to empower different security companies to find customers. Vendor i.e. Security Service companies sign up through web or app and provides corresponding licensing information, qualification which gets reviewed by admin and thereby approved or disapproved

System is designed with a sophisticated Vendor management dashboard whereby Vendors have complete visibility on the booking of different security services, corresponding rating given by customer, complete profile of security professionals as well as accounting information of the work done.

With a robust web platform and mobile app ProtectU provides several key features to users which includes easy booking service, reschedule service, booking extension, real time notification, GPS tracking, custom chatbots and much more.

This platform is provided on monthly, 6 month and 12 month subscription model for the Vendors and it’s free to user for end-users.
